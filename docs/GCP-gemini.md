## Start using GCP project  

|No.|Step|Description|
|--|--|--|
|01|Create playground project|It needs a project first. Follow [this guide](https://developers.google.com/workspace/guides/create-project) to create one|
|02|[Enable billing](https://docs.cloud.google.com/billing/docs/how-to/modify-project)|In order to use gemini models for exmaples it neeeds enabling billing, GCP offer $300 free credit in 90 days, it's enough for us to run playground scripts. If billing is disabled, "rate limit" errors will occur frequently.|
|03|Get API and Monitor cost|<ul><li>[Get an API key](https://aistudio.google.com/api-keys)<li>[Monitor cost](https://aistudio.google.com/usage?timeRange=last-7-days)|
